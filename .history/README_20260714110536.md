# GalzGames - Ingressos (backup)

Backup do site de venda/controle de ingressos. Este repositório contém um servidor Node simples e a pasta `public` com frontend.

Dependências e execução

1. Instalar dependências:

```bash
npm install
```

2. Rodar localmente (usa JSON local por padrão):

```bash
npm start
# ou
node server.js
```

Variáveis de ambiente recomendadas

- `PORT` — porta do servidor (padrão 3000)
- `MONGO_URI` — URL de conexão com MongoDB (opcional)
- `QR_SECRET` — segredo para geração de QR codes
- `ADMIN_SENHA` — senha administrativa (padrão presente no código)

Subir para o GitHub (substituir repositório perdido)

1. Crie um repositório no GitHub (pode usar o mesmo nome do anterior).
2. No diretório do projeto, execute:

```bash
git init
git add .
git commit -m "Backup: subir projeto GalzGames-Ingressos"
git branch -M main
git remote add origin GIT_URL_AQUI
git push -u origin main
```

Se quiser que eu faça o push direto, informe a URL do repositório `git` ou dê autorização para criar um novo repositório pela GitHub CLI.

Deploy em produção

Este projeto é uma aplicação Node (tem `server.js`). Para deploy use Render, Vercel (Serverless/Node support), Railway ou Heroku. Em plataformas como Render, defina `start` como `node server.js` e variáveis de ambiente (`MONGO_URI`, `QR_SECRET`).

Observações

- Os arquivos de dados em `data/*.json` são usados como fallback se o MongoDB não estiver configurado. Se quiser preservar dados sensíveis, remova ou limpe `data/` antes do push.

Sistema completo de geração e validação de ingressos online para eventos GalzGames.

## ✅ Funcionalidades

- **Página pública** com banner do evento personalizável
- **3 tipos de ingresso**: Normal, VIP, Meia-Entrada
- **Ingresso visual** com imagem de fundo, número único e QR Code
- **Validação** por número de ingresso
- **Painel Admin**: trocar imagem, editar dados do evento, ver vendas

## 🚀 Como rodar

### Requisitos
- Node.js 14+

### Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/galzgames-ingressos.git
cd galzgames-ingressos

# Rodar o servidor
node server.js
```

Acesse: **http://localhost:3000**

## 📁 Estrutura

```
galzgames-ingressos/
├── server.js           ← Backend Node.js (sem dependências)
├── package.json
├── data/
│   ├── tickets.json    ← Ingressos gerados (auto-criado)
│   └── config.json     ← Configurações do evento (auto-criado)
└── public/
    ├── index.html      ← Frontend completo
    └── images/
        ├── logo.png
        ├── banner.jpeg
        └── uploads/    ← Imagens enviadas pelo admin
```

## 🌐 Deploy no Railway / Render / Fly.io

1. Suba o repositório no GitHub
2. Conecte ao Railway ou Render
3. Defina `node server.js` como comando de start
4. Pronto! 🎉

## 🔌 API Endpoints

| Método | Rota | Descrição |
|--------|------|-----------|
| GET | `/api/config` | Configurações do evento |
| POST | `/api/config` | Atualizar configurações |
| GET | `/api/tickets` | Listar todos os ingressos |
| POST | `/api/tickets` | Gerar novo ingresso |
| POST | `/api/validate/:number` | Validar ingresso |
| POST | `/api/upload-bg` | Enviar imagem de fundo |
